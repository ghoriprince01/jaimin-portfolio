# Jaimin Raval – Portfolio Website

A professional portfolio website built with Node.js, Express, and EJS.

## Tech Stack
- **Backend**: Node.js + Express
- **Templating**: EJS
- **Frontend**: Vanilla CSS + JS (no frameworks)

## Getting Started

### Install dependencies
```bash
npm install
```

### Run the server
```bash
node server.js
```

Then open **http://localhost:3000** in your browser.

## Project Structure
```
jaimin-portfolio/
├── server.js           # Express server
├── views/
│   └── index.ejs       # Main HTML template
├── public/
│   ├── css/style.css   # All styles
│   ├── js/main.js      # Client-side JS
│   └── images/         # Profile photo
└── package.json
```

## Features
- Responsive design (mobile + desktop)
- Smooth scroll animations
- Contact form with server-side handling
- Dark/gold luxury theme
